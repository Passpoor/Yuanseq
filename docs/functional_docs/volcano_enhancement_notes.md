# 火山图功能增强说明

## 🎯 新增功能概览

### 1. 基因标签显示系统
- **自定义基因输入**：用户可以输入逗号分隔的基因列表
- **智能基因匹配**：支持SYMBOL和GeneID两种标识符
- **实时显示**：点击"显示自定义基因"按钮立即在图上显示

### 2. 基因标签样式控制
- **字体大小**：8-16像素可调
- **字体加粗**：可开关控制
- **标签颜色**：可自定义颜色
- **箭头指示**：带箭头的标签指向对应基因点

### 3. DeepSeek AI基因搜索
- **自然语言搜索**：输入基因功能描述（如"肿瘤抑制基因"）
- **自动填充**：搜索结果自动填入自定义基因输入框
- **API集成**：需要配置DeepSeek API密钥

### 4. 火山图导出增强
- **静态火山图**：包含所有自定义基因标签
- **高质量导出**：PNG (300 DPI) 和PDF格式
- **自定义尺寸**：可调整导出图片的宽度和高度

## 🔧 详细使用说明

### 自定义基因显示
1. 在"自定义显示基因"输入框中输入基因名称（逗号分隔）
   - 例如：`TP53, MYC, EGFR, CDK1`
2. 点击"显示自定义基因"按钮
3. 火山图上将显示这些基因的标签和箭头

### 基因标签样式调整
1. **字体大小**：拖动滑块调整标签字体大小 (8-16像素)
2. **字体加粗**：勾选"标签加粗"使字体变粗
3. **标签颜色**：点击颜色选择器修改标签颜色
4. **清除标签**：点击"清除自定义基因"移除所有标签

### DeepSeek基因搜索
1. 在"搜索基因描述"输入框中输入功能描述
   - 例如：`细胞周期调控基因`
   - 例如：`肿瘤抑制相关基因`
2. 点击"搜索相关基因"按钮
3. 系统将调用DeepSeek API并自动填充基因列表

### 配置DeepSeek API
1. 点击顶部菜单栏的"账户" -> "API配置"
2. 在"DeepSeek API密钥"输入框中输入您的API密钥
3. 点击"保存API密钥"按钮
4. 可点击"测试API连接"验证配置是否正确

**获取API密钥**：访问 https://platform.deepseek.com/api_keys

## 📊 技术实现细节

### 交互式火山图 (Plotly)
- 使用 `add_annotations()` 添加基因标签
- 支持箭头指向和悬停信息
- 实时响应所有样式调整

### 静态火山图 (ggplot2)
- 使用 `geom_text_repel()` 避免标签重叠
- 支持导出时保持所有自定义设置
- 高质量矢量图形输出

### API集成
- 使用 `httr` 包进行HTTP请求
- 使用 `jsonlite` 包处理JSON数据
- 错误处理和用户反馈

## 🎨 界面布局

火山图设置面板现在分为四个主要区域：

1. **基因标签设置**
   - 自定义基因输入
   - 显示按钮

2. **DeepSeek搜索**
   - 基因描述搜索
   - 搜索按钮

3. **标签样式设置**
   - 字体大小、加粗、颜色

4. **点样式设置**
   - 点大小、透明度

5. **坐标轴设置**
   - X轴范围、Y轴类型
   - 标签和标题大小

6. **颜色设置**
   - 上调/下调基因颜色

## 🔄 工作流程

1. **数据准备**：完成差异分析
2. **基因选择**：
   - 手动输入基因列表，或
   - 使用DeepSeek搜索相关基因
3. **样式调整**：调整标签和点的外观
4. **实时预览**：在交互式火山图上查看效果
5. **导出图片**：下载高质量图片用于论文发表

## 💡 使用技巧

- **批量操作**：可以一次性输入多个基因
- **智能匹配**：系统会自动匹配SYMBOL和GeneID
- **样式统一**：导出的静态图与交互图保持一致
- **错误处理**：未找到的基因会给出提示信息

这个增强功能使火山图更加灵活和实用，特别适合在论文中突出显示关键基因。
